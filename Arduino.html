Arduino Mobile Robot

IT Elective Project | Group Activity

Project Overview

A group activity using Arduino Uno to create a fully functional Mobile Robot (MOBOT). This project focused on integrating hardware components with C++ based Arduino programming to achieve autonomous or remote-controlled movement.

Through this workshop, we gained hands-on experience in circuit design, motor control logic, and collaborative engineering. The robot successfully demonstrated mobility and responsiveness during the final presentation.
