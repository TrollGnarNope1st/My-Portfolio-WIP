Quest Tracker (Capstone)

About the Project

Quest Tracker is a parent/child task monitoring application. It allows parents to assign tasks to their children and monitor their progress. Children are rewarded with physical rewards upon completion of tasks, encouraging responsibility and positive behavior.

Key Features

  • Task Assignment and Tracking
  • Reward System Integration
  • Real-time Progress Monitoring
  • User-friendly interface for both parents and children
