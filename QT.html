Quest Tracker

Capstone Project | 2024

About the Project

Quest Tracker is a parent/child task monitoring application designed to bridge the gap between household responsibilities and positive reinforcement. It allows parents to assign tasks to their children and monitor their progress in real-time.

Key Features

  • Task Assignment and Real-time Tracking
  • Reward System Integration for Physical Incentives
  • Interactive Progress Monitoring Dashboard
  • User-friendly Interface for Multi-generational Use
