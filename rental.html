DSite - Phone Rental

About the Project

DSite - Phone Rental is a robust device management system designed to streamline the process of renting smartphones. It features real-time monitoring, remote locking capabilities, and automated kiosk timers to enhance user experience and operational efficiency.

Admin Web Features

  • Real-time Fleet Dashboard to monitor device status, battery, and connectivity
  • Remote Hardware Control (Instantly lock/unlock screens and block settings)
  • Automated Rental Timer Management with an integrated daily earnings ledger
  • Live Screen Mirroring via WebRTC for remote viewing and troubleshooting
  • Granular App Management (Remotely hide or suspend specific apps per device)

Android Client Features

  • Hardware-level Kiosk Mode (Natively disables navigation gestures and buttons)
  • Bulletproof Background Watchdog to prevent bypassing or uninstallation
  • Deep System Integrations (Overlays, usage tracking, and battery optimization bypass)
  • Low-power background loop for offline survival and instant command execution
